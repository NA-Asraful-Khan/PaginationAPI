# Project Name

A Paginated API

## Getting Started

### Prerequisites

Ensure you have the following installed on your machine:

- Node.js (v14 or above)
- npm or yarn package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/NA-Asraful-Khan/PaginationAPI.git
   cd PaginationAPI
   npm install
   npm run start
   ```
